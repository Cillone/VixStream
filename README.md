# 🎬 VixStream

A sleek streaming app powered by **VixSrc** and the **TMDB API**.

---

## 🚀 Getting Started

### 1. Create a TMDB Account

Sign up here:
[The Movie Database (TMDB)](https://www.themoviedb.org/?utm_source=chatgpt.com)

### 2. Generate Your API Key

After creating your account:

* Click your **profile icon**
* Go to **Settings**
* Select **API**
* Click **Create API**

### 3. Connect VixStream

Copy your API key and paste it into **VixStream** to start streaming your favorite movies and shows.
